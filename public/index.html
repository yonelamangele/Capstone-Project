Users
HTTP Methods Path Description
GET /users Gets all the users
GET /users/:id Gets a single user by ID
POST /users/register Adds a user
PATCH /users/update/:id Edits a user info by ID
DELETE /users/delete/:id Deletes a single user by ID
Products
HTTP Methods Path Description
GET /products Gets all the products
GET /products/:id Gets a single product by ID
POST /products/addProducts Adds a product
PATCH /products/update/:id Edits a product info by ID
DELETE /products/delete/:id Deletes a single product by ID
Cart
HTTP Methods Path Description
GET /user/:id/carts Gets all the products added to the cart
POST /user/:id/cart Adds the product to the cart
PATCH /user/:id/cart/:id Updates the product added to the cart
DELETE /user/:id/cart Deletes products from the cart
DELETE /user/:id/cart/:id Deletes a specific product from the cart
